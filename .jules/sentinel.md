## 2026-03-05 - [DOM XSS in Print functionality]
**Vulnerability:** A DOM XSS vulnerability was identified in `src/components/Modals.tsx` and `labcontrol-spfx/src/webparts/labControlApp/components/Modals.tsx` where user-controlled variables (`item.name`, `item.lotNumber`, `item.id`, etc.) were directly interpolated into a `document.write` call when generating a print window.
**Learning:** `document.write` is highly dangerous if used with unsanitized input, even in a temporary or print-only window.
**Prevention:** Avoid `document.write` altogether if possible. If required, always ensure any dynamic variables interpolated into the HTML string are properly escaped (replacing `<`, `>`, `&`, `"`, `'` with their HTML entities).
