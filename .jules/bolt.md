## 2024-03-04 - [Inventory List Virtualization]
**Learning:** The application renders a large list of inventory items (`flatList`) using a native `map` loop inside `NativeList` component within `src/components/inventory/InventoryList.tsx`. The project already has `react-window` installed in `package.json` (`"react-window": "^1.8.11"`), and `react-virtualized-auto-sizer` (`"react-virtualized-auto-sizer": "^1.0.25"`). Adding virtualization to this list is an obvious performance improvement.
**Action:** Refactor `NativeList` in `src/components/inventory/InventoryList.tsx` to use `FixedSizeList` or `VariableSizeList` from `react-window` and `AutoSizer` from `react-virtualized-auto-sizer`.
